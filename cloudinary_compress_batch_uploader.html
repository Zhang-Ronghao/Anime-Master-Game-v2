Cloudinary 批量压缩并上传图片工具

本页面完全在浏览器本地运行。它会先用浏览器 Canvas 压缩图片，再把压缩后的文件上传到 Cloudinary。

不要填写 API Secret。浏览器直传只使用 Cloud Name 和 Unsigned Upload Preset。

1. Cloudinary 配置

如果 folder 报错，清空这里，改到 Cloudinary preset 的 Asset folder 里设置。

2. 上传前压缩设置

推荐：最大宽高 960，格式 WebP，质量 0.75–0.82。你的截图猜动画小游戏不需要 2K/4K 原图。
